# Quick Start - Decap CMS Admin Panel

## For Content Editors

### Accessing the Admin Panel

1. Go to: **https://psuigsa.github.io/admin/**
2. Click "Login with GitHub"
3. Authorize the application
4. Start editing!

**Unpublish Event:**
1. Open the event
2. Toggle "Published" to OFF
3. Click "Publish"
